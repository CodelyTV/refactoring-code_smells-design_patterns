# Product pricing: TypeScript

TypeScript codebase for the [Graduated Product pricing](../../README.md) exercise.

## Setup

```
# install dependencies
npm install
```

## Testing

### Unit Testing

```
npm run test
```

### E2E Testing

```
# run in dev mode on port 8080
npm run dev

# open Cypress to run tests on dev mode
npm run cy:open

# open Cypress on CLI
npm run cy:run
```

## Running the app

```
# install dependencies
npm install

# run in dev mode on port 8080
npm run dev

# generate production build
npm run build

# run generated content in dist folder on port 8080
npm run start
```

## Linting

```
# run linter
npm run lint

# fix lint issues
npm run lint:fix
```
